# 🧠 AI-Powered One-Day Outing Planner

## 📌 Overview

Planning a team outing often turns into long, unproductive group chats with no clear decision. This project solves that problem by providing an AI-powered planner that collects individual preferences and generates optimized outing plans.

---

## 🚨 Problem Statement

A team of 10 people wants to plan a one-day outing with the following constraints:

* Budget: ₹800 per person
* Distance: Within 40 km
* Food Preferences: Mix of vegetarian and non-vegetarian

The current planning process is inefficient, unstructured, and often leads to no conclusion.

---

## 💡 Solution

Build an AI-powered system where:

* Each team member inputs their preferences
* The system processes constraints and preferences
* It generates 2–3 optimized outing plans

Each plan includes:

* 📍 Recommended venue
* 🍽️ Food options (veg & non-veg)
* 💰 Estimated cost breakdown

---

## ⚙️ Features

* Preference-based input system
* Budget and distance optimization
* Smart recommendation engine
* Multiple outing plan suggestions
* Clear cost estimation

---

## 🛠️ Tech Stack (Suggested)

* Frontend: React.js
* Backend: Node.js / Express
* AI Logic: Python / OpenAI APIs
* Database: MongoDB

---

## 🚀 How It Works

1. Users enter preferences (food, budget flexibility, etc.)
2. System aggregates inputs
3. AI processes constraints
4. Generates 2–3 outing plans
5. Users choose the best option
